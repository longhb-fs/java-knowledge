## Yêu cầu

- **JDK**: Java 21 LTS (khuyến nghị)
- **IDE**: IntelliJ IDEA Community (miễn phí) hoặc VS Code + Extension Pack for Java
- **Thời gian**: 2-3 giờ/ngày

## Cách học hiệu quả

1. **Đọc lý thuyết** - Hiểu concept trước
2. **Gõ code** - Không copy/paste, tự gõ để nhớ
3. **Làm bài tập** - Mỗi ngày có bài tập thực hành
4. **Debug** - Học cách debug để hiểu code chạy thế nào
